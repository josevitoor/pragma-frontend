Bem-vindo ao PRAGMA – Plataforma Rápida para Geração de Módulos Automatizados.

O PRAGMA foi desenvolvido para simplificar e acelerar a criação de código CRUD, permitindo gerar, de forma rápida e padronizada, os componentes e funcionalidades de um sistema.

Módulo de Configurações de Estrutura do Projeto

Neste módulo, você define a estrutura e caminhos de um projeto específico, como pastas e arquivos, tanto da API quanto do Cliente. Ele serve como um modelo de estrutura que poderá ser reutilizado.

Módulo de Configurações de Caminhos

Aqui você informa o caminho físico do projeto na sua máquina. Cada caminho está vinculado a uma configuração de estrutura previamente cadastrada, garantindo que os arquivos sejam gerados exatamente nos locais corretos.

Módulo de Configurações de Conexões

Aqui você pode gerenciar as conexões de banco de dados e os caminhos dos projetos. Isso facilita a geração do código, permitindo selecionar uma conexão e um caminho pré adicionado sem a necessidade de digitar manualmente.

Módulo de Geração de Código

Este módulo permite gerar código CRUD de maneira prática e rápida, seguindo três etapas principais:

  • Selecionar ou digitar a conexão do banco de dados.
  • Selecionar ou digitar o caminho e estrutura do projeto
  • Configurar a geração do código: tabela, nome da entidade, campos de listagem, filtros, etc.

Com esses módulos, o PRAGMA garante a geração de código consistente, seguindo padrões técnicos e de arquitetura, economizando tempo e evitando tarefas repetitivas.
